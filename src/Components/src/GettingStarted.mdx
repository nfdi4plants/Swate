# Getting Started

## Install

### JavaScript

#### Install component library

`npm i @nfdi4plants/swate-components`

#### Import the components you want to use:

```jsx

import '@nfdi4plants/swate-components/style.css' // Import minified tailwind and daisyui style
import { TermSearch } from '@nfdi4plants/swate-components'

<TermSearch
    onTermSelect={(term) => console.log(term)}
    term={undefined}
    parentId='TEST:1298'
/>

```

### .NET

#### Install with **Femto**

`dotnet femto install Swate.Components <path/to/.fsproj>`

#### Install **Manually**:

Install [nuget package](https://www.nuget.org/packages/Swate.Components).

```
<PackageReference Include="Swate.Components" Version="1.0.0" />

paket add Swate.Components
```

Install npm dependencies

```
npm i @floating-ui/react@^0 @tanstack/react-virtual@^3 @uidotdev/usehooks@^2
```

#### Import the components you want to use:


```fsharp
open Fable.Core.JsInterop

Swate.Components.SwateStyle.import

Swate.Components.TermSearch.TermSearch(
    (fun term -> Browser.Dom.console.log term),
    None
)
```
